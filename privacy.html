隐私政策

本应用尊重并保护所有用户的个人隐私。

1. 信息收集

本应用不会主动收集用户的个人身份信息。用户在使用 API Key、模型地址、书源、对话或相关配置功能时，相关数据仅用于实现应用功能。

2. 信息使用

用户填写的配置信息仅用于本地功能调用或用户主动配置的第三方 API 服务。本应用不会将用户数据出售、出租或用于广告追踪。

3. 第三方服务

如果用户配置了第三方 API、模型服务或书源服务，相关请求可能会发送至用户自行配置的第三方服务。第三方服务的数据处理规则以其自身隐私政策为准。

4. 数据存储

应用中的配置数据可能保存在用户设备本地。用户可以通过删除应用或清除配置来移除相关数据。

5. 未成年人隐私

本应用不会有意收集未成年人的个人信息。

6. 隐私政策更新

我们可能会根据应用功能变化更新本隐私政策。更新后的政策将在本页面发布。
