常见问题

1. API Key 如何配置？

请进入「模型」页面，在 API Key 输入框中填写你的服务商 API Key。

2. API 地址如何配置？

请进入「模型」页面，在 API 地址中填写对应服务商的接口地址。

3. 书源为空怎么办？

请进入「书源」页面，点击右上角 + 添加书源。

4. 如何清除配置？

可以在应用内删除相关配置，或卸载应用后重新安装。
